# Kanban Task Manager

Ung dung quan ly cong viec theo mo hinh Kanban board.

## Tinh nang

- Dang nhap/Dang ky voi ma hoa SHA256
- Phan quyen Manager va Employee
- Kanban board 3 cot: Todo, Doing, Done
- Drag and Drop de chuyen trang thai task
- Tim kiem, loc theo do uu tien
- Sap xep theo deadline
- Xuat bao cao CSV
- Quan ly thanh vien

## Cong nghe

- Framework: .NET Framework 4.8
- UI: Windows Forms
- Database: MySQL 8.0
- Security: SHA256 password hashing

## Cai dat

1. Clone repository
2. Chay file SQL: firstapp/Database/full_database_setup.sql
3. Mo firstapp.sln trong Visual Studio
4. Restore NuGet Packages
5. Nhan F5 de chay
